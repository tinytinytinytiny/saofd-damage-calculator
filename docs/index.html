Damage Calculator

Input your character information below to calculate damage. The “Attack” section can be shortcutted by inputting your total attack (seen in-game) into Base Attack, and setting all other values to 0.

Categories labeled “Unconfirmed” need testing in regards to their position in the formula. Some information, such as character Base Attack, Character Rank Attack, Buff values, and Passive Skill values cannot be seen in-game, but can be found throughout this site.

Depending on which screen you view your attack stat in-game, the value may be rounded. However, on most screens, the mathematical ceiling is used so any decimal is pushed up to the nearest integer, i.e. 1000.2 → 1001.

If you notice any errors, please feel free to point them out in our discord.
